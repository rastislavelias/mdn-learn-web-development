# Challenge: Mobile‑first Responsive Layout

This challenge is part of the [MDN Learn Web Development Tutorials](https://developer.mozilla.org/en-US/docs/Learn_web_development). It focuses on practicing mobile‑first responsive design by building layouts that adapt from small to large screens following the project specifications.

## Description

The goal of this project is to design and implement a responsive layout using a mobile‑first workflow. This includes:

- Starting with styles for small screens and progressively enhancing with media queries.
- Using flexible units (%, rem, vw), responsive images, and fluid typography.
- Applying CSS Grid and Flexbox for adaptable component and page layouts.
- Ensuring touch‑friendly controls, readable line lengths, and accessible navigation.
- Testing across different viewport sizes and devices.

For more details, see the [project brief](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/CSS_layout/Mobile-first_challenge#project_brief).

This project is for learning purposes and helps build practical skills in mobile‑first, responsive layout techniques.
