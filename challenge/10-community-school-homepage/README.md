# Challenge: Community School Homepage

This challenge is part of the [MDN Learn Web Development Tutorials](https://developer.mozilla.org/en-US/docs/Learn_web_development). It focuses on practicing text styling and typesetting by designing a clear, readable homepage for a community school according to the project specifications.

## Description

The goal of this project is to create a homepage with strong typographic structure and readable content. This includes:

- Establishing a clear typographic hierarchy with headings, subheadings, and body text.
- Controlling line length, spacing, and rhythm for better readability.
- Choosing appropriate font sizes, weights, and line heights.
- Using semantic HTML for content structure and CSS for typographic styling.
- Ensuring the design is accessible and responsive across screen sizes.

For more details, see the [project brief](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Text_styling/Typesetting_a_homepage#project_brief).

This project is for learning purposes and helps build practical skills in text styling and typesetting for the web.
